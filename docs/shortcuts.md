# WNim Hotkeys

## Navigation

| Key | Action |
|-----|--------|
| Ctrl+Space | Autocomplete |
| Ctrl+E | File menu (left side) |
| Ctrl+F | Find text |
| Ctrl+G | Go to line |
| Home | Start of line |
| End | End of line |
| PgUp | Page up |
| PgDn | Page down |
| Arrow Left | Move cursor left |
| Arrow Right | Move cursor right |
| Arrow Up | Move cursor up |
| Arrow Down | Move cursor down |

## File Operations

| Key | Action |
|-----|--------|
| Ctrl+N | New file |
| Ctrl+O | Open file |
| Ctrl+S | Save |
| Ctrl+R | Save as |
| Ctrl+Q | Quit |

## Editing

| Key | Action |
|-----|--------|
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+C | Copy line |
| Ctrl+X | Cut line |
| Ctrl+V | Paste |
| Ctrl+K | Delete line |
| Tab | Indent |
| Shift+Tab | Unindent |

## Tabs

| Key | Action |
|-----|--------|
| Ctrl+T | New tab |
| Ctrl+W | Close tab |
| F1 | Previous tab |
| F2 | Next tab |

## Information

| Key | Action |
|-----|--------|
| Ctrl+A | File statistics |

## Additional Functions

| Key | Action |
|-----|--------|
| Ctrl+E | Open file menu (↑↓ to navigate, Enter to open, Esc to close) |
| Ctrl+L | Load plugin |
| Ctrl+U | Unload plugin |
| Ctrl+P | List plugins |

## File Menu Notes

- Press `Ctrl+E` to toggle file menu on the left
- Menu shows all code files in current directory
- Use `↑`/`↓` arrows to navigate
- Press `Enter` to open selected file in new tab
- Press `Esc` or any other key to close menu
- Menu is small (12 files visible, 30 chars wide)

## General Notes

- Ctrl+K supports hold for deleting multiple lines
- Ctrl+O opens file in a new tab
- Ctrl+Q requires confirmation on unsaved changes
- Autocomplete works only when suggestions are available
- Lua plugin system is supported
